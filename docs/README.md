# 技巧

## 可以做

- 手头上有offer（如果是大公司更好，大公司的中低薪容易拿）

> 那些 6k-10k 的，但是，如果要拿，很容易拿到(但根本不会去接这种offer)，拿到后，手头上就有几个 offer ，心理安稳，面试起来也从容，也可以讨价还价，说，我手头上也有几个 offer了，所以我想知道贵公司的 xx

- 会的题目装做想了一下才会

> 这样显得你水平高，基本问的都会，如果按正常答，面试管根据你会的，会提更难得，那整个面试有好些题目回答不上


- 分2种，一种只为了拿offer,一种是真正为了找工作

- 表现的自己很厉害

> 比表现自己不行好，其次，你需要做到的是，不要太在意这个机会，哪怕你真的很在意。

- 提前准备说辞，为什么从上一家公司离职，提前说几遍

> [常规的问题](https://www.zhihu.com/question/20192678/answer/294654747)
